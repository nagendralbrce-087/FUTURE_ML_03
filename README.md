# Resume Screening System

## Project Overview

This project uses Natural Language Processing (NLP) and Cosine Similarity to rank resumes based on a job description.

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* TF-IDF Vectorization
* Cosine Similarity
* Matplotlib

## Features

* Resume Ranking
* Job Description Matching
* Similarity Score Calculation
* Resume Screening Visualization

## Dataset

A sample resume dataset containing candidate skills and qualifications.

## Methodology

1. Load resumes from the dataset.
2. Define a job description.
3. Convert text into numerical vectors using TF-IDF.
4. Calculate similarity scores using Cosine Similarity.
5. Rank resumes based on similarity scores.

## Results

Top Resume Match Score: 79.06%

## Conclusion

The system successfully ranks resumes according to their relevance to the job description, helping recruiters identify the most suitable candidates.
